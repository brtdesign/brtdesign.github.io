Instant UI and user comprehension

Slowing things down to speed things up

2nd August 2014

by Admin

A fast, responsive user interface is essential to a successful modern web app. I recently worked on part of an interface that, whilst not particuarily slow, suffered from a few 100ms delay between the user taking action and the UI updating.

After updating the way the scripts worked, the interface became to fast, and users were prone to missing changes. To overcome this, I took a series of steps make the UI feel extremely fast, but aid the users understanding of whats happened.
